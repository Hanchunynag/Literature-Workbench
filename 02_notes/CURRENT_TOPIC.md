# 当前课题

## 研究主题
GNSS拒止环境下基于LEO机会信号的导航与定位方法研究

## 当前主线
- Iridium + Orbcomm 多星座融合
- 伪距率 / Doppler 类观测定位
- robust WLS / HVCE
- 异常观测抑制与鲁棒赋权
- 轨道误差补偿 / 残差学习

## 当前关键问题
- 多星座异构观测如何做动态赋权
- 弱信号环境下多普勒提取误差如何传递到定位层
- 伪距不足时混合观测定位的可观性边界
- 星历误差与 light-time 建模对定位性能的影响

## 本批文献后的即时判断
- 多星座融合定位已经有较明确的 Iridium + ORBCOMM 工程验证基础，可直接作为主线参考文献。
- 鲁棒 HVCE 是当前最贴近本课题“多星座 + 鲁棒赋权”交叉点的方向，应优先吸收其加权框架。
- 弱信号条件下的 Doppler 提取质量，可能是连接“信号处理层”和“定位解算层”的关键切入点。
- Doppler-aided WLS 说明在观测不足场景下，混合观测解算值得继续跟踪，但需区分“融合 LEO 导航系统”和“纯 SOP”两类应用假设。
